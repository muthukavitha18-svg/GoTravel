📌 Project Description
This project is a travel website homepage called GoTravel, built using only HTML5 and CSS3 — without using any JavaScript frameworks or libraries.
The goal of this project is to design a modern travel website interface inspired by real-world platforms, while improving core front-end development skills such as layout design, responsive design, and styling techniques.
The project includes a sticky navigation bar, hero search section, destination cards, hotel and restaurant listings, and a structured footer.
________________________________________
🎯 Objective
•	Build a structured webpage using semantic HTML 
•	Create responsive layouts for all screen sizes 
•	Design card-based UI components 
•	Implement hover effects and transitions 
•	Practice CSS Flexbox and layout techniques
